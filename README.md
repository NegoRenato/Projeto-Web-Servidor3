Esta e a terceira parte do projeto da disciplina de web-servidor, desenvolvido por Renato Antonio e Flavio Cesar

A terceira parte consiste em, ou refatorar o codigo dos projetos anterior para o larave, ou criar uma api para aplicação dos projetos anterior
(Optamos por refatorar o codigo para laravel).

Passo a Passo da instalação.

requisitos: e necessario ter o php na versão 8+, e o xampp instalado para rodar a base de dados.

tutorial: deve baixar o conteudo do projeto do meu git, e sem seguidar descompactar a pasta que vira compactada, e logo após
mover a pasta compactada para a dentro da pasta htdocs, que fica dentro da pasta xampp. E depois deve importar o arquivo
da base de dados que esta dentro do projeto (nome e 'laravel.sql'), para o phpmyadmin.

Executando o projeto.

Para executar o projeto deve, executar o seguinte comando no terminal 'composer run dev', ao executar ele vai oferecer a url para voce
acessar o projeto, e la voce podera testar as funcionalidades.
